# Code of Conduct

## Our Pledge

We want AFKBOT to be an open, respectful, and technically serious project.
Contributors, maintainers, and community members are expected to act in ways
that make participation safe and productive for everyone.

## Expected Behavior

- Be respectful and professional in public and private discussions.
- Assume good intent, but be precise when challenging ideas or code.
- Focus feedback on technical content, behavior, and project goals.
- Help keep discussions actionable and relevant to the project.

## Unacceptable Behavior

- Harassment, intimidation, or personal attacks.
- Discriminatory language or conduct.
- Publishing private information without permission.
- Repeated disruption, bad-faith trolling, or abusive review behavior.

## Scope

This code of conduct applies to project spaces, issue trackers, pull requests,
discussions, and other community channels related to AFKBOT.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code,
issues, and other contributions that violate this code of conduct.

## Reporting

Report serious violations privately to the project maintainer through the
official AFKBOT channels listed at [afkbot.io](https://afkbot.io). Security
vulnerabilities should still follow the process in [`SECURITY.md`](SECURITY.md).
